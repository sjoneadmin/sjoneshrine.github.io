<product_requirements>
The user initially requested a donations goal tracker with milestones, operable on GitHub, deployable as a static site on GitHub Pages. Key features included manual donation tracking, custom milestone amounts with descriptions/rewards, and a visual progress indicator (logo filling with color).

The application, *Sjòne Shrine Inc. Donations Goal Tracker*, now focuses on collecting donations for supporting emerging future female leaders in virtual athletic performance. It includes E-Sports coaching, recreational therapy, and gear.

**Implemented Features & Current State:**
*   **Visual Progress Indicator:** A custom leaf logo that progressively fills with a coral color (#FE6F5E) as donations increase, with each of its 7 leaves representing progress segments.
*   **Milestone System:** Five phases (Phase I: Research & Visioning, Phase II: Architectural Planning, Phase III: Ground Breaking, Phase IV: Infrastructure & Equipment, Phase V: Buildout & Programming) with Roman numerals and detailed descriptions. Progress is tracked against these milestones.
*   **Donation Form:** A simplified frontend-only form for entering donation amount, name, email, and message.
*   **Payment Options:** Currently features two integrated payment methods:
    *   **Bank:** Using a provided NBKC Bank payment link () and a dynamically displayed QR code, described as FDIC Secure transaction network.
    *   **PayPal:** Using a provided PayPal.me link ().
*   **Confirmation & Receipt:** Upon payment button click, the user is redirected to the external payment site, followed by an automatic confirmation page with a detailed donation summary, thank you message, and a downloadable receipt (TXT file).
*   **Data Persistence:** Uses browser  for all donation and progress data, as the application has been converted to frontend-only for static site deployment.
*   **Design:** Follows a motion-first approach with smooth transitions, bold gradients (coral theme), depth through shadows, and clear typography.
</product_requirements>
